Data Analysis Skills

Descriptive Statistics
  • Used for exploration and examining data trends
  • Dealing with differnet data types
  • Measures of center
  • Measures of spread
  • Looking into data shape(s) and variable trends
  • Spotting Outliers, and dealing with them
Inferential Statistics
  • Used for drawing conclusion(s) about whole population used in dataset
  • conclusion(s) are made using Sampling Distribution mainly
  • Bootsampling using NumPy/Pandas libraries
  • conculsion(s) are made using <Confidence Interval> which means they are true to large extent
Probability

  • Measures used are:
    1. Binomial Distribution
    2. Conditional Probability
    3. Bayes Rule
  • Looking at distribtion of binomial variables using Binomial Distribution
Hypothesis Testing
